Lab 10-"Practice printing a string

Create a string using .ascii.  Print that string on the screen.  Exit gracefully to the operating system at the end of your program.

Use these constants:

.equ    STDIN,0
.equ    STDOUT,1
.equ    READ,3
.equ    WRITE,4
.equ    EXIT,1
.equ    SUCCESS,0

Use this code to print:

movl    $promptSiz, %edx
movl    $prompt, %ecx
movl    $STDOUT, %ebx
movl    $WRITE, %eax
int     $0x80

File: print.asm
